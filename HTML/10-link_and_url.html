URLs

Uniform Resource Locator

scheme://host.domain:port/path/file_name?query_string#anchor

  1. Scheme – Protocol (http, ftp, mailto, file)
  2. Host – Domain host (default for HTTP is www)
  3. Domain – Website name (e.g., google.
  4. Port – Server connection port (default HTTP = 80)
  5. Path – Resource path in the server
  6. File Name – Name of the HTML/file
  7. Query String – Parameters for web apps (?key=value&...)
  8. Anchor – Specific location on the page (#id)
